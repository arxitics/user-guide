Features

  • Advanced search interface to find articles
  • Analysis, visualization, and recommendations
  • Subscribe to specific categories, keywords, and authors
  • Collect interesting articles and reviews
  • Post your reviews and make comments
  • Revise and extend metadata fields
  • Share your unpublished documents
  • Real-time and collaborative chat for the community

Search Engine

  • Powered by MongoDB
  • More than 1 million documents
  • As many as 50+ metadata fields
  • 17 single field indexes and 1 text index
  • Supports full-text search, but sometimes slow
  • Accepts JSON to construct complicated query expressions

Search Syntax

  • Full-text search
  • Keyword-style search
  • JSON string

Redirects

  • arXiv identifier
  • Digital object identifier (DOI)
  • Only one result is found for your query

Full-text Search

  • Match any of the search terms
  • Search for an exact phrase
  • Exclude documents that contain a term

Keyword-style Search

  • Specify equality condition
  • Equality match on fields within an embedded document
  • Specify AND conditions on multiple fields

Search Array Fields

  • Specify OR conditions
  • Specify AND conditions
  • Specify OR as well as AND conditions
  • Match a specific element at a particular index
  • Match a field in the embedded document using the array index
  • Match a field in the embedded document without specifying the index

Query Operators

  • Match all values that are not equal to a specified value
  • Match any of the values specified in an array
  • Match none of the values specified in an array
  • Match values that are greater than a specified value
  • Match values that are less than a specified value
  • Match values that are greater than or equal to a specified value
  • Match values that are less than or equal to a specified value
  • Join query clauses with a logical OR
  • Join query clauses with a logical AND
  • Join query clauses with a logical NOR
  • Match documents that have the specified field
  • Select documents if a field is of the specified type
  • Select documents if the array field is a specified size
  • Select documents where values match a specified regular expression
  • Match documents that satisfy a JavaScript expression
  • Perform full-text search

Date Range

  • Sepecify the date in the form of yyyy-mm-dd or part of it
  • Sepecify the date with date-range and a special word
  • Sepecify the date with date-from and date-to

Cursor Methods

  • Order documents in the result set
  • Set the starting point of the result set
  • Limit the number of documents to return

More Search Examples

  • Mixing of full-text search and keyword-style search
  • Specify how many results are shown per page
  • Specify the page number of search results

Search Tips

  • The field names of arrays are plurals
  • Keyword-style queries on most fields are case-sensitive
  • Always specify categories to filter eprints
  • Avoid full-text search when possible
  • Specify a limit if there are maybe too many results
  • Try to specify more conditions if no results are found
